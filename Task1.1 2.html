Project Description

The mirror cleaning robot is an autonomous device designed to climb and clean mirrors using brushless
motors, a propeller, and N20 gear motors. Controlled by an Arduino UNO and Bluetooth module, it
effectively removes stains and debris from vertical surfaces, providing an efficient and convenient
cleaning solution.

Project Description

The project aims to create a solar energy charger using Arduino, incorporating a solar tracker to
maximize sunlight exposure. It harnesses solar energy, charges batteries, and prevents overcharging
through an Arduino microcontroller. The system's efficiency and reliability are enhanced by
continuously adjusting the solar panel's angle for optimal sun tracking.
